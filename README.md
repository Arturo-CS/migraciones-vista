# Laboratorio - Vista de Datos

## Description

Desarrollo de vista de los datos que provienen de la BD.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
